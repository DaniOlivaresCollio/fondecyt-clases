  • Skilled: ciuo08 == 51, 54, 61, 62, 71:75, 81 con media completa o más (educ >= 5)

  • Unskilled: ciuo08 == 52, 63, 82, 83, 90+

  • Unskilled: ciuo08 == 51, 54, 81 sin información educ (educ == NA)
